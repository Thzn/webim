Thresholding

In many vision applications, it is useful to be able to separate out the regions of the image corresponding to objects in which we are interested in. Thresholding often provides an easy and useful way to perform this segmentation, on the grounds of the different intensities or colors in the foreground and background areas of an image.

threshold(tvalue,image)
tvalue : value between [0,255]
image : Gray image

var img2;
function preload() {
img2=loadImage("car.jpg");
}

function setup() {
createCanvas(img2.width*2,img2.height);
var newim=rgbToGray(img2);
displayGray(threshold(122,newim));
image(img2,img2.width,0);
}
