Sharpening

Image sharpening can be used to extract features of the smoothened image. Firstly, it uses the inbuilt edge detection methods to identify the edges of the image, and then it uses the original image for sharpening. The resulting image will be more sharpened depending on the edge detector used.

var img2;
function preload(){
img2=loadImage("coin.jpg");
}
function setup() {
createCanvas(img2.width*2,img2.height);
var newim=rgbToGray(img2);
var edges=edge(newim);

for(var x=0;x newim.pixels[x]=newim.pixels[x]+edges.pixels[x];
}
displayGray(newim);
image(img2,img2.width,0);
}
