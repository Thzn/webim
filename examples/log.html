Log Transform

Log transformation is one of the simplest point processing methods used in image preprocessing. In frequency domain image processing, log transformation is used to identify the hidden features of the image. The left figure below shows the effect of log transformation.

threshold(tvalue,image)
tvalue : Threshold value between [0,255]
image : Gray image

var img2;
function preload(){
img2=loadImage("car.jpg");
}
function setup() {
createCanvas(img2.width*2,img2.height);
var newim=rgbToGray(img2);
displayGray(threshold(122,newim));
image(img2,img2.width,0);
}
