Inversion

Image inversion is a point processing method used to identify the features of the image. Most of the time inversion is called as the negative of an image. Some times in fields like medical imaging it is better to identify image as an negative for more good illustration of the image sample. Bellow images shows the inversion of an brain scan image. (Left-Invert, Right-Original)

invert(image)
image: gray image

var img2;
function preload(){
img2=loadImage("coin.jpg");
}
function setup() {
createCanvas(img2.width*2,img2.height);
var newim=rgbToGray(img2);
displayGray(invert(122,newim));
image(img2,img2.width,0);
}
