Rotation

A Simple method that implements image rotation. The method uses heavy floating point arithmetics on each pixel. Since there can be rounding errors when using floating point arithmetic, the images can introduce noise patterns. Therefore, a smoothing method can be used to reduce the noise of the image. For instance, the Median filter might be the best approach that can be used to reduce this type of salt-pepper noise.

rotate(image,x,y,deg)
image : Gray image
x,y : Coordinates of rotation origin
deg : Angle of rotation

var img2;
function preload(){
img2=loadImage("coin.jpg");
}
function setup() {

createCanvas(img2.width*2,img2.height);
var newim=rgbToGray(img2);
displayGray(rotate(newim,newim.width/2,newim.height/2,-45));
image(img2,img2.width,0);
}
